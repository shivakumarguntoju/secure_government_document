üë§ User Workflow

Simple and intuitive user experience

1. Registration & Authentication

User creates account with email, password, and Aadhaar verification. Email verification is automatic.

2. Document Upload

Upload documents (PDF, DOC, Images up to 5MB), categorize by type (Aadhaar, PAN, Passport, etc.), and add descriptions.

3. Document Management

View documents in grid layout, preview files, edit metadata, download originals, and search across all documents.

4. Sharing & Collaboration

Share documents with family members via email or Aadhaar, set view-only or download permissions, track shared documents.

5. Profile & Activity

Manage profile information, view complete activity history, monitor security events, and review document statistics.
